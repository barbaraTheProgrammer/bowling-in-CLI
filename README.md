# bowling-php

This is game in php, which is working in console. Rules of bowling are easy. You are trying to knock down as much pins as you can. If you get 10 pins in one try you get full bonus - point from next dwo rolls. If you get 10 pins in whole turn you get half bonus - points from next one roll. Program, is showing general score after every turn and showing scoretable at the end. If you are good enough to get 10 pins at the end of the game, you get extra roll!
Game is based on two classes Game and Frame (turn). Program works on objects.
